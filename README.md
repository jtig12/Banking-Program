# Banking Program

## Description

    The essence of this project is to act as a banking app where the users can view there acct balance, make deposit and withdraw there money.

## Steps taken to develop the project

1. First step is to create a function to show balance, deposit money , and withdraw the money

2. Second step is to create a loop canstantly asking for which function the user wants to perfom on the app and calling out the function.

3. Then create various if conditions where the functions will be called if a certain condition in this case a number from 1-4.

4. Then we begin workig on the show balance function.

5. Next we work on the deposit function to put money into the bank and change the available balance.

6. Next we work on the withdraw function to remove money from the bank and make it reflect on the new available balance.

7. Then we break out of the loop if we no longer depositing, withdrawing or checking our balance.
